# qr code #
一个简单的二维码生成系统。

戳进来：[二维码生成系统](http://mayuko.cn/lab/qr/)

**食用方法：**

1. 输入内容（如果生成网址链接请包含http://）
2. 输入大小（1-20）
3. 点击确定
4. 右键保存

## API ##
这个api可方便生成指定字符串或网址的二维码，并且，你可以自定义size。
